OctoPrint APT repository

Debian packages related to OctoPrint.

Signing key: apt.octoprint.org/octoprint.gpg.key

GitHub: github.com/OctoPrint/apt.octoprint.org

Usage

RaspberryPi OS/OctoPi

curl -s --compressed "https://foosel.github.io/apt.octoprint.org/octoprint.gpg.key" | gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/octoprint.gpg > /dev/null
sudo curl -s --compressed -o /etc/apt/sources.list.d/octoprint.list "https://foosel.github.io/apt.octoprint.org/octoprint-bullseye-rpi.list"
sudo apt update
